> Sistema de produção de conteúdo para **Mercado Livre Afiliados** via vídeos curtos (Reels / TikTok / Shorts).
> Cada produto passa pelo fluxo completo: **Análise → Roteiro → Prompts Veo3 → Vídeo → Publicação**.

---

## Fluxo de produção

```
01-Produtos-em-Analise  →  02-Roteiros-Prontos  →  03-Prompts-Veo3  →  04-Videos-Publicados
        ↓                          ↓                       ↓
   Escolher produto           Adaptar roteiro         Gerar no Veo3
   validar nicho              para o criador          montar no editor
```

## Como usar este sistema

1. **Escolha um produto** em `01-Produtos-em-Analise/produtos.md`
2. **Abra o roteiro** correspondente em `02-Roteiros-Prontos/`
3. **Gere as cenas** com os prompts em `03-Prompts-Veo3/` no Veo3
4. **Monte o vídeo** no editor seguindo a ordem das cenas + adicione os textos das notas de edição
5. **Publique** e registre em `04-Videos-Publicados/`
